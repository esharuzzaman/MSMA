# MSMA
Emf Prediction
These are the matlab code behind the constitutive modeling of MSMA described in the folowing article:
M. E. Emu, “A New Constitutive Model for Ferromagnetic Shape Memory Alloys and Prediction of Electromotive Force,” Northern Arizona University, 2020.
The published article can be found in the following:
https://doi.org/10.1088/1361-665X/acb2a1
